To use the argoverse-hd dataset, you need to first download the data from the following link and put the zipped contents into "DL-reweighting/dataset-ninja". For the binary case you need to run the "Create datasets for binary testing" cell in preprocessing.ipynb.

All unbiased data: https://drive.google.com/file/d/1wmih3VXyHetpDBZsfO46IC1cMub0b1Zt/view?usp=sharing
